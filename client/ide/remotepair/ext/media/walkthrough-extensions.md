# Install the AI extensions

Xpair works best with three extensions from Open VSX:

- **Claude Code** (`anthropic.claude-code`)
- **ChatGPT** (`openai.chatgpt`)
- **Open Remote - SSH** (`jeanp413.open-remote-ssh`)

They are installed automatically the first time Xpair starts. If any are
missing, run **Xpair: Install AI Extensions** from the Command Palette (or
the button above). Reload the window if prompted.
