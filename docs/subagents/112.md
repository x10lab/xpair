# 112

| ID | 동작 | 예상 | 현행 | 플래그 |
|---|---|---|---|---|
| 112 | 첫 실행 온보딩에서 사용자가 Enter manually를 선택해 Connect 단계로 들어옴 | 수동 host 입력, Tailscale 안내, SSH 키 안내, Check connection으로 분기 | 동일 | continue |
| 1120 | 사용자가 수동 연결 화면에서 닫기 또는 이전 이탈을 선택함 | 온보딩 완료 없이 종료 | 동일 | terminate |
| 1121 | Tailscale 상태 확인 중 화면이 표시됨 | 안내만 대기하고 host 입력은 가능해야 함 | 동일 | continue |
| 1122 | Tailscale Ready 안내가 표시됨 | tailnet host 이름 입력을 권장하고 연결 판단은 SSH probe로 진행 | 동일 | continue |
| 1123 | Tailscale 설치됨, 미실행 안내가 표시됨 | Tailscale 실행 또는 일반 SSH host 입력을 안내하고 연결 판단은 SSH probe로 진행 | 동일 | continue |
| 1124 | Tailscale 미설치 안내가 표시됨 | Tailscale 설치 또는 reachable SSH host 입력을 안내하고 연결 판단은 SSH probe로 진행 | 동일 | continue |
