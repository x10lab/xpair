Client ↔ Host 교차 연결

If host accept → Client Waiting for Permission grant
host가 Broadcasting에서 client를 수락하면, client의 권한/수락 가드가 풀립니다.
Provides Folder Structure Information → Client Folder mappings
host가 폴더 구조를 제공해 client가 매핑을 구성할 수 있게 합니다.
